How to run self test on the 33120A Waveform Generator.

  1. Press and hold the [Shift] key while turning the power on.
  2. When when you here the 2nd beep release the shift key.
  3. The test will run and give a pass or fail indication on the display.
  4. This test will not diagnose a faulty output fuse which has a symptom of no output. If this is the case, please use the instant message box to notify us of the station and room that has the problem.
