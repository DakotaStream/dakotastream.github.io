How to use the HP54603B as a Spectrum Analyzer

  1. Push [Setup] button
  2. Push [Default Setup] soft button next to screen
  3. Push [+ -] button
  4. Push [Function 2 On] soft button
  5. Push [Functuion 2 Menu] soft button
  6. Push [Operation FFT] soft button
  7. Push [1] button, to see just the FFT

How to adjust Cent Freq

  1. Push [Cent Freq] soft button
  2. Rotate the knob closest to the [Cursors] button to change the value

How to adjust Freq Span

  1. Push [Cent Freq] soft button
  2. Rotate the knob closest to the [Cursors] button to change the value
